# Scripts Node.js

Este repositorio contiene varios scripts útiles desarrollados en Node.js para diferentes tareas como conversión de datos, descargas y migraciones.

## 📂 Estructura de carpetas

- **conversor_excel_json/** → Convierte hojas de cálculo Excel a JSON.
- **conversor_json_excel/** → Convierte archivos JSON a Excel.
- **conversor_json_rips/** → Procesa y convierte datos RIPS en formato JSON.
- **descarga_documentos_magaly/** → Automatiza la descarga de documentos.
- **descarga_nuevaeps/** → Descarga de archivos desde nuevaeps.
- **manual_chague_payjoy/** → Guías y scripts relacionados con PayJoy.
- **migracion_data_fija_dinamica/** → Scripts para migración de datos fijos y dinámicos.

## 🚀 Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Ren-jdcastro/script_node.git
   cd script_node
#
npm ci

Cada carpeta contiene su propio script y en algunos casos su propio package.json.
Entra a la carpeta correspondiente y ejecuta el script según sus instrucciones.
